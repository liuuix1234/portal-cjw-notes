# portal-cjw-notes

本仓库用于归档与发布多个独立的 HTML 页面。这些页面以笔记、工具或信息展示为目的，不绑定任何特定域名或网站。

## 项目简介

这是一个以“门户”形式组织的 HTML 页面集合。每个页面均为独立文件，可单独访问，无需依赖后端服务。适合用于展示、分享或临时托管静态内容。

## 目录说明

```
.
├── index.html          # 主入口页面（若存在）
├── pages/              # 存放各独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
└── README.md           # 本文件
```

- `index.html`：可选的主页，可用于导航至其他页面。
- `pages/`：核心目录，所有独立 HTML 页面均放置于此。

## 页面归档说明

- 每个 HTML 文件均为完整、自包含的页面，不依赖外部资源（或资源已内联）。
- 文件名应具有描述性，便于识别内容主题。
- 页面之间无强制关联，可独立使用。

## 维护说明

- 新增页面：直接在 `pages/` 目录下添加 `.html` 文件即可。
- 更新页面：请保持文件结构清晰，避免破坏已有链接。
- 删除页面：确保无其他页面引用后，可安全移除。
- 欢迎提交 Pull Request 或 Issue 来完善内容。

## 使用方式

1. 克隆仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等静态托管服务发布。
